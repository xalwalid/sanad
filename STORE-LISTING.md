## Data safety (Play)  &  App Privacy (Apple)

Phase-1 answer is the simplest and strongest: **no data collected, no data shared**.

Play "Data safety" form:
- Does your app collect or share any user data? → **No**
- Is all data encrypted in transit? → N/A (no data leaves the device)
- Do you provide a way to request data deletion? → Data is on-device; user can delete their journey in-app or by uninstalling.

Apple "App Privacy":
- Data Not Collected. (No tracking, no analytics SDK, no ads.)
- The photo-library permission is used only to save the achievement card the user explicitly creates; images are not uploaded.

> Keep this true. If we later add analytics or the community backend (Phase 3), both forms MUST be updated before that build ships.
